# Omnifood

A responsive marketing website for Omnifood — a fictional AI-powered meal subscription service that creates personalized weekly meal plans and delivers them through partner restaurants and cooks.

## About

Omnifood combines an AI-driven meal planning concept with a subscription delivery model. Users select dietary preferences (vegetarian, vegan, keto, paleo, gluten-free, and more), and the AI generates a tailored weekly meal plan, which gets cooked and delivered by local partners.

## Features Showcased

- Responsive multi-section landing page (Hero, How It Works, Diets, Testimonials, Pricing, Features, CTA, Footer)
- Pricing plans (Starter / Complete)
- Diet filter showcase
- Customer testimonials and photo gallery
- Sign-up form for a free sample meal
